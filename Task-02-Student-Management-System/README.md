# Task 02 - Student Management System

## Overview

The Student Management System is a console-based Java application developed as part of the Java Programming Internship at IncodeVision.

This application allows users to manage student records efficiently through a menu-driven interface.

## Features

* Add Student Records
* View All Students
* Update Student Information
* Delete Student Records
* Store Student Name, Roll Number, and Marks
* Menu-Driven Program
* Object-Oriented Programming (OOP)

## Technologies Used

* Java
* ArrayList
* OOP Concepts
* VS Code

## Functionalities

### Add Student

Users can add a new student by entering:

* Roll Number
* Name
* Marks

### View Students

Displays all student records stored in the system.

### Update Student

Allows modification of existing student details.

### Delete Student

Removes a student record using the roll number.
